# Customer Churn Analysis

## Project Overview
This project analyzes customer churn behavior using Excel-based data analysis techniques.
The goal of the project is to identify patterns behind customer cancellations and suggest strategies for improving customer retention.

## Tools Used
- Microsoft Excel
- Microsoft Word
- GitHub

## Dataset
IBM Telco Customer Churn Dataset

## Analysis Performed
- Churn distribution analysis
- Contract type vs churn
- Monthly charges analysis
- Internet service analysis

## Key Findings
- More customers stayed than churned.
- Month-to-month customers churn more than yearly contract customers.
- Customers with higher monthly charges are more likely to churn.
- Fiber optic users showed higher churn.

## Recommendations
- Offer discounts for long-term contracts.
- Provide loyalty rewards to long-term customers.
- Create personalized offers for high-risk customers.
- Reduce pricing issues through flexible plans.
